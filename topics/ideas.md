# Topic Bank

Ideas for LinkedIn content. Mark as used when published.

---

## How to Use This File

Format: `- [ ] Topic | Hook: "..." | Pillar: X | Format: Y`

**Pillars**: educational, storytelling, authority-jacking
**Formats**: story, insight, question, list, observation

When planning weekly content:
1. Select 4 topics balancing pillars and formats
2. Mark selected topics with `[P]` (planned)
3. After posting, mark with `[x]` and add week number
